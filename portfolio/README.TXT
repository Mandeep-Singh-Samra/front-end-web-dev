Landing Page Template Instructions

1. Edit `index.html` to change:
   - Headline text
   - Feature titles and descriptions
   - Contact form placeholders

2. Use your own images in the 'images' folder and update HTML `img` tags if added.

3. Customize colors in `style.css` if needed:
   - Hero background
   - Button colors
   - Text colors

4. To make it live:
   - Upload to GitHub and enable GitHub Pages
   - OR drag the folder to Netlify.com

Thanks for using this template!
